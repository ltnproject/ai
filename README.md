# 🔑 How to Get Your Own API Key (Groq)

## 🚀 Step 1 — Open Groq
Go to: https://console.groq.com  
Sign in with Google or GitHub.

---

## 🧠 Step 2 — Create API Key
1. Open **API Keys**
2. Click **Create Key**
3. Copy the key

Example:  gsk_xxxxxxxxxxxxxxxxx
---

## 📋 Step 3 — Use It in DrkChat
1. Go to sidebar → **Your API Key**
2. Paste your key
3. Done

---

## ⚡ Benefits
- No 15 message limit
- Faster responses
- More stable
- No shared usage issues

---

## 🔒 Security (Important)
- Never share your API key

---

## ❗ Common Errors
- Key doesn’t start with `gsk_`
- Extra spaces
- Invalid or revoked key
